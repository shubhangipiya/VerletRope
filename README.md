# VerletRope

An interactive typography experiment built with p5.js.

A paragraph of text sits on screen. As you drag your mouse 
through it, letters peel off one by one and fall to the ground 
with realistic physics — gravity, bounce, and rotation.

## Built with

- p5.js
- Vanilla JavaScript
- HTML Canvas

## What I learned

- Verlet physics simulation
- Canvas coordinate systems
- Particle systems and arrays
- Real-time mouse interaction
- Letter-by-letter text layout
